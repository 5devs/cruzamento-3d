Nodes e Fields
  • Nós (nodes) são os blocos básicos de construção do mundo VRML.
    O nome ou tipo do nó indica o objeto que ele descreve
    (shapes, lights, sounds, groups, sensors, etc.)


  • Todo nó tem um tipo (Shape, PointLight, Box, ...), um par de chaves, e campos opcionais


    • Box { }


  • Campos (fields) definem os atributos de um nó


  • Todo campo tem um nome, um tipo e um valor default


  • Os campos são opcionais, e seu valor default é utilizado caso não sejam especificados


    • Sphere { radius 2 } Box { size 2 3 1 } Shape { geometry Box { } }
