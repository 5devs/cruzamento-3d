Shape
  • Shape é o nó que descreve a geometria e a aparência de um objeto


    • Shape { appearance ... geometry ... }
  • Para descrição de geometria, VRML fornece alguns nós básicos como Sphere, Cylinder, Box, Cone e outros nós de geometria mais complexa


  • Além da geometria, um Shape também possui propriedades, como material e textura, que definem seu aspecto. Estas propriedades são descritas no campo appearance , com um nó Appearance
    • Shape { appearance Appearance { material ... texture ... ... } geometry ... }
