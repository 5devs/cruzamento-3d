Reduzindo Arquivos
  • Se o objeto é usado mais de uma vez, é mais eficiente dar um nome ao objeto com DEF e, depois, referenciá-lo com USE.


  • Devem ser prototipados os objetos que são usados frequentemente, com pequenas variações. O browser fica livre para otimizar as partes que não mudam.


  • Textos não devem ser especificados como polígonos.


  • Nós como Box,Cone, Cylinder, Sphere, rExtrusion e ElevationGrid fornecem um meio compacto para descrição de objetos com muitos polígonos.


  • Devem ser usadas as normais que são geradas automaticamente.


  • Devem ser eliminados espaços em branco no arquivo VRML.


  • Os números reais devem ter poucas casas decimais.


  • Devem ser usados arquivos comprimidos. Para texturas, de preferência, devem ser usados JPEG ou PNG; para movies, MPEG.
