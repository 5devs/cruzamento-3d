Mundos Compartilhados
  • Outra extensão em estudo para VRML é a criação de mundos compartilhados, mundos onde vários usuários interagem.


  • Prover acesso a vários usuários simultaneamente significa que os browsers de todos os usuários precisam ficar informados sobre o estado do mundo.


  • Se vários usuários estão conectados ao mundo, é necessário criar geometrias (avatar) para representar cada um deles.
