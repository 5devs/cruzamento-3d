Tendências a Curto Prazo
  • EAI (External Authoring Interface) permite que uma applet escrita em Java, em um página HTML envie eventos para o mundo VRML. A applet Java também pode ser notificada quando um nó da cena enviar eventos.


  • Um problema para VRML é a falta de hardware barato que ofereça desempenho razoável para explorar os mundos virtuais. Fabricantes de computadores têm sentido o potencial de aplicações 3D e vêm oferecendo hardware gráfico a preços menores.


  • Encontra-se em desenvolvimento um formato binário para arquivos VRML.O formato binário permite a criação de arquivos menores para download mais rápido.
