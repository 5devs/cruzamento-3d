MovieTexture
  • O nó MovieTexture permite mapear uma textura animada para um objeto.


  • Os formatos suportados são: MPEG1-Systems e MPEG1-Video.


  • Além de campos para especificar a textura a ser mapeada, o nó possui campos que permitem controlar a exibição do filme: velocidade, horas de início, hora de fim e indicador de loop da exibição.


    • MovieTexture { url "movie.mpg" loop TRUE speed 1.0 }
