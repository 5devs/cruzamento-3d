Fog
  • O nó Fog define uma região com visibilidade reduzida, para simular neblina ou fumaça.


  • O browser mistura as cores do fog com as cores dos objetos exibidos. Quanto mais distante do obsevador estiver o objeto, mais sua cor se confunde com a do fog.


  • O background não é afetado pelo nó Fog. Para dar maior realismo à cena, deve ser criado um background com a cor do fog.


  • Como o fog limita a visão do observador, ele reduz o trabalho de modelagem de objetos 3D e pode melhorar o desempenho do mundo VRML.
