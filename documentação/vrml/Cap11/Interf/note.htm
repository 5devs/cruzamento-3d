Declarando a Interface do Script
  • Para um programa em JavaScript, informe o script no campo url do nodo Script, campos e eventIns/eventOuts

DEF Pular Script {
field SFFloat alturaPulo 3.0
eventIn SFFloat set_fraction
eventOut SFVec3f value_changed
url "javascript: . . ." }
  • A url pode ser o código (programa) do Script ou um endereço para este
