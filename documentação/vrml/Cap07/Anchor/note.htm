Anchor
  • O nó Anchor cria um hyperlink com outras mídias (página HTML, imagem, mundo VRML).


  • O nó Anchor cria um grupo.
    • Cada nó desse grupo é exibido.

    • Ao ser apontado um objeto definido como nó filho de Anchor, o hyperlink é ativado.

  • Uma descrição pode ser associada ao nó Anchor.
