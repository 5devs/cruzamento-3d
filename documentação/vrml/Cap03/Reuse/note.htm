Reuso
  • Se um nó é utilizado várias vezes em uma cena, é possível dar-lhe um nome na primeira vez em que ele é descrito e, depois, fazer referência a ele, cada vez que se for usá-lo.


  • O uso de DEF e USE torna o arquivo VRML menor e rápido de carregar (melhorando a performance ), além de mais legível e fácil de modificar

  • Os nomes têm que ser únicos em um arquivo


  • Nomes são necessários para animação
