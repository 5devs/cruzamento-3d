Transform Transform { translation ... rotation ... scale ... children [...] }
  • Um nó Transform é um nó de agrupamento (group node) que cria um sistema de coordenadas local para o nós filhos.


  • A ordem de uma série de transformações definidas dentro de um mesmo nó Transform é fixa:

    • mudança de escala
    • rotação
    • translação


  • Para mudar a ordem das transformações, é necessário aninhar nós Transfom.
