Inline
  • O nó de agrupamento Inline lê seus filhos de qualquer endereço na Web (URL)


  • Um dos gargalos de desempenho ocorre quando um arquivo VRML é carregado. Se todos os objetos são incluídos no arquivo principal, o browser tem que carregar todos eles

    Com Inline, o browser não carrega imediatamente o nó. Em vez disso, ele vai ao URL procurar os dados e, enquanto isso, mostra uma caixa no local onde o objeto deve aparecer
