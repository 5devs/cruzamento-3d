Billboard
  • O nó de agrupamento Billboard cria um grupo com um sistema de coordenadas especial, que é automaticamente rotacionado para ficar de frente para a câmera


  • É útil, por exemplo, para inclusão de imagens 2D que devem ficar sempre de frente para a câmera (textos, árvores, etc.)


  • O nó Billboard é um meio econômico de acresentar realismo à cena
