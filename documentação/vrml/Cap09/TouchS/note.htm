TouchSensor
  • O nó TouchSensor monitora nós de geometria que pertencem ao seu grupo.


  • O nó gera eventos quando o cursor aponta para uma geometria por ele monitorada.


  • Eventos gerados:
isActive envia TRUE/FALSE quando o botão do cursor é pressionado
isOver envia TRUE/FALSE quando o cursor é movido para dentro/fora de uma geometria
hitPoint_changed envia o ponto no objeto indicado pelo cursor
hitNormal_changed envia a normal à superfície do objeto no ponto indicado pelo cursor
hitTextCoord_changed envia a coordenada da textura no ponto indicado pelo cursor
touchTime envia a hora que o botão do cursor é solto
