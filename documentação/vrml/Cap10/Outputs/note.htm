Usando as Saídas dos Temporizadores

  • O evento isActive :
    • Envia TRUE quando o temporizador inicia
    • Envia FALSE quando o temporizador termina
  • O evento de saída time :
    • Envia o tempo absoluto
  • O evento de saída fraction_changed :
    • Envia valores de 0.0 até 1.0 durante um ciclo
    • Volta para 0.0 no início de cada ciclo
