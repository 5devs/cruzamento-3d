Convertendo Tempo para Posição

  • Para animar a posição de um objeto, você informa:
    • uma lista de posições-chave para um determinado caminho
    • os tempos nos quais se deve estar a cada posição
  • O nodo interpolador de posição converte um tempo de entrada em uma posição de saída
  • Quando um tempo está entre duas posições-chave, o interpolador calcula uma posição intermediária
